# 💸 Rishabh's Budget Tracker

A responsive, single-page expense tracking web app built with vanilla HTML, CSS, and JavaScript — no frameworks required.

> Built to understand frontend development concepts like state management and DOM manipulation — the same thinking behind React.js components.

## Features
- Add income and expense transactions with Singapore-specific categories
- Live balance, income and expense summary cards
- Delete individual transactions or clear all
- Data persists via localStorage — survives page refresh
- SGD currency formatting and Singapore date format

## What I Learned
- Managing application state in JavaScript and syncing it to the DOM
- How localStorage mirrors client-side database behaviour
- Component-style thinking — each transaction card is rendered from data, the same mental model behind React
- Responsive layout using CSS Grid

## Tech Stack
HTML · CSS · JavaScript
